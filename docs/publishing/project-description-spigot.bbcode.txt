[SIZE=7][B]BedrockGUI v2[/B][/SIZE]
[SIZE=5][B]Bring native Bedrock Edition forms to your Java server![/B][/SIZE][/CENTER]

BedrockGUI is a powerful utility for [B]Paper[/B], [B]Velocity[/B] and [B]Bungeecord[/B] that allows server owners to create native Bedrock-style interactive forms. By integrating with Floodgate, it provides a seamless UI experience for Bedrock players while remaining manageable via simple YAML configurations.

[SIZE=6][B]✨ Key Features[/B][/SIZE]

[SIZE=5][B]🖼️ Versatile Form Types[/B][/SIZE]
[LIST]
[*][B]Simple Forms[/B]: Button-based menus. Support for custom images via URLs, textures, or player names.
[*][B]Modal Forms[/B]: Simple Yes/No confirmation dialogs.
[*][B]Custom Forms[/B]: Complex data entry with Text Inputs, Sliders, Dropdowns, and Toggles.
[*][B]Java Menus[/B]: Early support for traditional chest-based GUIs.
[/LIST]

[SIZE=5][B]⚡ Unified Action System[/B][/SIZE]
Execute complex logic using a modern curly-brace syntax. Chain actions together to create guided player flows.
[LIST]
[*][B]Commands[/B]: Run commands as the player or the console.
[*][B]Economy[/B]: Integrate with your economy plugin for balance checks and transactions.
[*][B]UI Feedback[/B]: Send titles, action bars, and colorized messages.
[*][B]Inventory[/B]: Manage player items (give/remove/check).
[*][B]Flow Control[/B]: Use [ICODE]delay[/ICODE], [ICODE]random[/ICODE], and [ICODE]conditional[/ICODE] actions to build intelligent menus.
[*][B]BungeeCord[/B]: Send players to different servers via plugin messaging.
[/LIST]

[SIZE=5][B]🔌 Integrations[/B][/SIZE]
[LIST]
[*][B]Floodgate[/B]: Native support for Bedrock player detection.
[*][B]PlaceholderAPI[/B]: Use any PAPI placeholder in your titles, button text, or action values.
[*][B]YAML Configuration[/B]: No need to recompile; edit your menus in real-time.
[/LIST]

[SIZE=6][B]⌨️ Commands[/B][/SIZE]
[LIST]
[*][ICODE]/bedrockgui[/ICODE] — Administrative control (Permission: [ICODE]bedrockgui.admin[/ICODE])
[*][ICODE]/bedrockgui reload[/ICODE] — Reloads all configurations.
[*][ICODE]/bedrockgui open <menu> [player][/ICODE] — Opens a specific form for a player.
[/LIST]
